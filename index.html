About These Tools

This collection contains single-file HTML tools built with the assistance of Claude Code. Each tool is completely self-contained with inline CSS and JavaScript, requiring no build process or external dependencies (except for CDN-hosted libraries like Prism.js for syntax highlighting). All tools run entirely client-side in your browser.
