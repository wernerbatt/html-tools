# HTML Tools Collection

Miscellaneous HTML+JavaScript tools built with the help of [Claude Code](https://claude.com/claude-code).

All tools are single-file HTML applications that run entirely in your browser, with no build process or server dependencies required.

## Text Tools

- [Text Minifier](https://wernerbatt.github.io/html-tools/text-minifier.html) — Compress and minify text by removing unnecessary whitespace, line breaks, and tabs
- [URL Encoder/Decoder](https://wernerbatt.github.io/html-tools/url-encoder.html) — Encode and decode URLs and URI components for safe transmission and API integration

## API and Data Tools

- [API Prettifier](https://wernerbatt.github.io/html-tools/api-prettifier.html) — Format and validate JSON/XML API responses with syntax highlighting, line numbers, and copy-to-clipboard functionality

## About

This repository contains a collection of single-file HTML tools. Each tool is completely self-contained with inline CSS and JavaScript, making them easy to use, share, and deploy.

### Key Features

- **No build process** — Just open the HTML file in a browser
- **Self-contained** — All CSS and JavaScript is inline
- **Client-side only** — Everything runs in your browser
- **No dependencies** — Except for CDN-hosted libraries (like Prism.js)
- **Portable** — Easy to copy, share, and host anywhere

## Development

These tools are created with the assistance of [Claude Code](https://claude.com/claude-code), demonstrating rapid prototyping and development of useful utilities.

### Adding a New Tool

1. Create a new `.html` file in the repository root
2. Make it self-contained with inline CSS and JavaScript
3. Update `index.html` to add it to the tool list
4. Update this README with a description

### Guidelines

- Keep tools focused on a single purpose
- Include clear instructions within the tool's UI
- Use modern, clean design
- Ensure responsive layout for mobile devices
- Add helpful error messages and validation
